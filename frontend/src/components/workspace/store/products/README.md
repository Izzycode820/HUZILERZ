# Store Products Components

## Components to be implemented:

### 1. ProductsList.tsx
- Display all products with filtering/search
- Product status (active/inactive)
- Quick actions (edit, delete, duplicate)

### 2. AddProduct.tsx
- Product creation form
- Image uploads
- Inventory management
- Vendor source selection (owned/affiliate)

### 3. ProductCard.tsx
- Individual product display component
- Product thumbnail, title, price, stock
- Quick action buttons

### 4. BulkImport.tsx
- CSV/Excel file upload
- Import preview and validation
- Bulk product creation

### 5. ProductSync.tsx
- Sync products to deployed templates
- Template selection and sync status
- Bulk sync operations
